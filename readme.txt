Shows the number of users on the site

== Description ==

Plugin to show the current active visitors on the website, the daily, monthly and all time visitors

FUNCTIONS

- Plugin counter of current users on site
- Ability to change title
- Change the background and text colors

== Installation ==

1. Select 'Add new' in the 'Plugins' menu to find and install the current Visitors Right Now plugin.
2. Activate the plugin.
